Projeto 2:

O objetivo deste projeto é recriar a estrutura e o layout do site The New York Times utilizando as tecnologias web HTML, CSS e JavaScript. Este projeto tem como finalidade aprimorar as habilidades de desenvolvimento front-end, incluindo a criação de layouts responsivos, a estilização de componentes e a adição de interatividade através de scripts.

Projeto 4:

O Validador de Cadastro é uma aplicação simples desenvolvida para garantir que os formulários de cadastro estejam preenchidos corretamente antes de enviar os dados. Utilizando HTML para estruturar o formulário, CSS para estilização e JavaScript para validação dinâmica, esta aplicação oferece uma experiência amigável e intuitiva para os usuários.

Play

Projeto 5:

O Mapa Interativo é uma aplicação web desenvolvida para proporcionar uma experiência similar ao Google Maps, permitindo aos usuários explorar locais, obter direções e descobrir informações detalhadas sobre pontos de interesse. Utilizando tecnologias como HTML, CSS, JavaScript e o famoso XAMPP para desenvolvimento local, esta aplicação oferece uma interface intuitiva e responsiva para navegação geográfica.

Projeto 7:

EO Chat Simples em Tempo Real é um projeto exemplar que demonstra a comunicação assíncrona em um ambiente de servidor estático, utilizando tecnologias fundamentais como HTML, CSS e JavaScript. Este projeto permite aos usuários enviar e receber mensagens em tempo real, simulando a interação entre cliente e servidor localmente usuários.
